# Kriss Beauty UGC Website

## Deploy to Netlify in 3 steps (FREE, forever)

1. Go to **netlify.com** and sign up free
2. Drag and drop this entire folder onto the Netlify dashboard
3. Your site is live instantly at a free URL like `krissbeauty.netlify.app`

## Custom Domain (optional)
- In Netlify, go to Domain Settings
- Add your custom domain (e.g. thekrissbeauty.com)
- Free SSL included automatically

## Files
- `index.html` — Main site structure
- `style.css` — All styling (pink/beige/brown palette)
- `script.js` — Animations, typewriter, counters, FAQ, form

## To update content
Edit `index.html` in any text editor (Notepad, TextEdit, VS Code)
Then re-drag the folder to Netlify to redeploy

## Contact Form
Currently opens the user's email client with a pre-filled message to thekrissbeauty@gmail.com
To get form submissions in your inbox without opening email, sign up for a free Netlify Forms account and add `data-netlify="true"` to the form tag.
